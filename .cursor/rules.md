# Cursor Rules

1. Always try and test what you've generated.
2. Only ask for permission to delete files. Don't ask for permission to add files.
3. Only ask for permission to run commands that delete files. Don't ask for permission to add/move files.
4. Only add comments if explcitly asked.
5. Always add tests for the code you're writting.
6. Always update docs/API.md if a new endpoint is created.
7. Always update the docs/iterative-notes.md
